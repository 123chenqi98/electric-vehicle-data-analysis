# 📋 项目经历 - 新能源汽车保值率归因分析与体验特征挖掘

---

## 📌 项目一：基于LLM的新能源汽车保值率预测模型

### 🏷️ STAR法则版本

#### 📍 S - 情境 (Situation)

在新能源汽车市场竞争日益激烈的背景下，二手车保值率成为影响消费者购车决策和品牌忠诚度的关键因素。传统保值率预测主要依赖车辆基础属性（车龄、里程、品牌），但**缺乏对用户真实体验的关注**。团队获批一项数据分析课题，需要从73,761条车辆登记数据与13万条车主评论中，挖掘影响保值率的深层驱动因子。

**行业痛点**：
- 现有保值率模型精度不足（行业平均R²约0.65-0.70）
- 非结构化评论数据未能有效利用
- AI特征工程在垂直领域应用尚属空白

---

#### 🎯 T - 任务 (Task)

作为项目独立负责人，我需要：

1. **构建全流程数据Pipeline**：从数据采集、清洗、特征工程到建模的端到端解决方案
2. **创新性引入LLM特征**：使用大模型从非结构化评论中提取隐性体验指标
3. **实现高精度预测**：模型R²目标≥0.80，超越行业基准
4. **提供可解释性**：通过SHAP值归因，为业务决策提供数据支撑

---

#### ⚡ A - 行动 (Action)

##### 🔹 第一阶段：数据治理与预处理

```
技术栈：Python | Pandas | NumPy

• 处理73,761条车辆数据，识别并处理8个关键字段的缺失值
• 使用IQR方法检测并移除价格、续航里程等字段的异常值
• 构建自动化数据质量监控Pipeline，实现数据验证自动化
• 建立数据血缘追踪机制，确保特征可追溯
```

##### 🔹 第二阶段：AI特征工程（核心创新）

```
技术栈：阿里云百炼API | LangChain | 正则表达式

• 设计Prompt模板，引导LLM从评论中提取7个隐性维度：
  ✓ 电池焦虑指数（用户对电池衰减的担忧程度）
  ✓ 智能驾驶满意度（对辅助驾驶功能的评价）
  ✓ 内饰品质感知（用料、做工、设计的综合评分）
  ✓ 充电便利性（充电速度、桩覆盖率的主观体验）
  ✓ 续航满意度（实际续航与标称续航的吻合度）
  ✓ 售后服务体验（维修、保养、客服的综合评价）
  ✓ 性价比感知（价格与产品价值的匹配程度）

• 开发批量处理脚本，单次调用可处理100条评论
• 实现失败重试机制，确保API调用稳定性>99.5%
• 构建特征质量评估体系，验证LLM提取的一致性
```

##### 🔹 第三阶段：特征工程与建模

```
技术栈：Scikit-learn | XGBoost | LightGBM | Label Encoding

• 对分类特征（品牌、型号、动力类型）进行Label Encoding
• 对数值特征进行Z-score标准化处理
• 使用XGBoost构建回归模型，采用GridSearchCV进行超参调优
• 对比实验设计：
  ├─ 基线模型：仅传统特征（车龄、里程、品牌等）
  ├─ AI增强模型：传统特征 + LLM提取的7个体验特征
  └─ 消融实验：验证每个AI特征的独立贡献度
```

##### 🔹 第四阶段：SHAP可解释性分析

```
技术栈：SHAP | Matplotlib | Seaborn

• 计算全局SHAP值，识别特征重要性排名
• 生成个体预测解释，支撑业务决策
• 构建"电池焦虑-保值率"因果分析框架
• 输出可视化报告，包含特征贡献度热力图、依赖图
```

## 📎 面试高频问题准备

### Q1: 这个项目的创新点是什么？

**参考回答**：
> 最大的创新点是将LLM应用于垂直领域的特征工程。传统做法是对评论做情感分析（正面/负面），但我们设计的Prompt可以引导模型输出结构化的体验评分。具体来说，我从13万条评论中提取了7个维度的隐性指标，比如"电池焦虑指数"，这个特征在后续建模中显示出高达95%的正向贡献，证明了用户体验数据对保值率的预测价值。

---

### Q2: 如何保证LLM提取特征的可靠性？

**参考回答**：
> 我设计了多层质量保障机制：
> 1. **Prompt优化**：通过Few-shot Learning提供示例，确保输出格式一致
> 2. **输出校验**：用正则表达式验证评分范围（1-10），自动过滤异常值
> 3. **一致性检验**：随机抽取5%的样本进行人工标注，与LLM输出对比，准确率>85%
> 4. **业务验证**：将LLM特征与已知业务规律对比，如Tesla品牌智驾满意度应高于行业平均

---

### Q3: AI特征到底带来了多少提升？

**参考回答**：
> 从模型指标看，加入AI特征后：
> - R²从0.75提升到0.82，提升了9.3%
> - MAE从$4,890降低到$4,500，降低了8%
>
> 从业务价值看，SHAP分析显示7个AI特征中，有3个进入TOP 5重要性排名，证明了用户体验数据对保值率的预测价值是不可忽视的。
